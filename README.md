
# QCPUC Foundation

> A standard library for an Objective C flavour compiled by [QCPU CLI](https://github.com/QSmally/QCPU-CLi).

## Compiler types

* `InstanceType` (uppermost class)
* Literals:
    - `StringLiteralType` (`"foo"`)
    - `UIntLiteralType` (`24`)
* `ReferenceType` (usermode pointer)
